# 🎯 START HERE - BETCI Application

## 🚀 Quickest Way to Run

### Just double-click this file:
```
RUN-APP.bat
```

It will:
- ✅ Check if MongoDB is running
- ✅ Start MongoDB (if needed)
- ✅ Start the backend server
- ✅ Open the dashboard in your browser

---

## 📁 Important Files

| File | What It Does |
|------|--------------|
| `RUN-APP.bat` | Starts everything (MongoDB + Backend) |
| `STOP-APP.bat` | Stops everything |
| `CHECK-STATUS.bat` | Shows what's running |
| `RUN-APP.ps1` | PowerShell version of RUN-APP.bat |
| `STOP-APP.ps1` | PowerShell version of STOP-APP.bat |
