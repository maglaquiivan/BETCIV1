🔧 Fix 431 Error

Clear session storage to resolve "Request Header Fields Too Large" error

⚠️ The Problem

Your browser's localStorage contains large base64-encoded profile pictures, causing HTTP 431 errors when making requests to the server.

✅ The Solution

Click the button below to clear the problematic data. Your login session will be preserved, but you'll need to reload your profile picture (it will load automatically from the server).
